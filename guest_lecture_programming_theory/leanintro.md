# Lean
- A functional programming language and proof assistant.
- Large mathematical library (Mathlib)
- Created by Leonardo de Moura while employed by Microsoft Research
- Open source and has a large community (lastest version: Lean 4)

- Installation: https://lean-lang.org/install/
  - VSCode
  - Lean4 Extension
- Also available as a web page: https://live.lean-lang.org/
