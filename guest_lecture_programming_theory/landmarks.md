# Landmark Projects
- L4.verified (G. Klein et al., 2009): functional correctness of the seL4 microkernel (about 8700 lines of C) -- using Isabelle
- CompCert (X. Leroy et al., 2009): a formally verified compiler for (almost all of) the C language -- using Rocq.
- Flyspeck (T. Hales et al., 2014): a formal proof of the Kepler conjecture -- using Isabelle and HOL Light
- FLT (WIP, [github.com/ImperialCollegeLondon/FLT](https://github.com/ImperialCollegeLondon/FLT)): a formal proof of Fermat's last theorem -- Lean 4
