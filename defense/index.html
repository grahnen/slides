Event-Driven Programs

Handlers communicate via:

  • Message passing
  • Shared Memory

Handlers may access shared memory, and send messages to the mailboxes of other handlers

When a process is not busy, it can fetch a message from its mailbox and start executing it

We study executions of programs running on this model
